# 辞書ディレクトリ

このディレクトリに英単語→カタカナ読み辞書ファイルを配置してください。

## 📚 Bilingual Kana Dictionary

このアプリケーションでは、**bilingual_kana_dictionary.json** を使用して、
英単語を日本語のカタカナ読みに変換します。

### 特徴

✅ **シンプル**: JSON形式の辞書ファイル  
✅ **セットアップ不要**: MeCabなどの外部ツールが不要  
✅ **軽量**: 小さなファイルサイズ  
✅ **カスタマイズ可能**: 簡単に単語を追加・編集できる  

---

## 📁 辞書ファイル

### bilingual_kana_dictionary.json

英単語とそのカタカナ読みをマッピングしたJSON辞書です。

**ファイル形式**:
```json
{
  "hello": "ハロー",
  "world": "ワールド",
  "app": "アプリ",
  "game": "ゲーム",
  "user": "ユーザー"
}
```

**現在の収録語数**: 約150単語

---

## 🔧 辞書のカスタマイズ

### 単語の追加

`bilingual_kana_dictionary.json` をテキストエディタで開き、
以下の形式で単語を追加してください：

```json
{
  "既存の単語": "カタカナ読み",
  "new": "ニュー",
  "word": "ワード"
}
```

**注意事項**:
- 最後の行にはカンマ `,` を付けない
- UTF-8形式で保存
- JSONの形式が正しいか確認（オンラインJSONバリデーターを使用）

### 複数形の扱い

英語の複数形も別途定義することを推奨します：

```json
{
  "game": "ゲーム",
  "games": "ゲーム",
  "user": "ユーザー",
  "users": "ユーザー"
}
```

## 🔍 辞書の探索順序

アプリケーションは以下の順序で辞書を探索します：

1. `<実行ファイルのディレクトリ>/dic/bilingual_kana_dictionary.json`
2. `<実行ファイルのディレクトリ>/mascots/english_readings.json`（フォールバック）
3. 組み込みのデフォルト辞書（約40単語）

## 📖 使用例

### 入力テキスト
```
Hello, this is a test app for game users.
```

### 変換後
```
ハロー, this is a テスト アプリ for ゲーム ユーザー.
```

辞書に登録されていない単語（`this`, `is`, `a`, `for`）は、
アルファベット読み（`ティーエイチアイエス`など）に変換されます。

## 💡 推奨される辞書の拡張方法

### よく使う単語を追加

アプリケーションで頻繁に使用する単語を追加すると、


## ⚠️ トラブルシューティング

### 辞書が読み込まれない

**原因**: ファイルパスが間違っている、またはJSON形式が不正

**対処法**:
1. ファイルパスを確認: `dic/bilingual_kana_dictionary.json`
2. JSONの形式を確認（オンラインバリデーターを使用）
3. UTF-8形式で保存されているか確認
4. デバッグログで辞書の読み込み状況を確認

### 一部の単語が変換されない

**原因**: 辞書に単語が登録されていない

**対処法**:
1. 辞書ファイルに単語を追加
2. 大文字小文字は区別されません（`Hello` も `hello` も同じ）
3. アプリを再起動して辞書を再読み込み

---

## 📝 注意事項

- 辞書ファイルは大文字小文字を区別しません
- 辞書に登録されていない単語は、アルファベット1文字ずつ読みに変換されます
- 辞書ファイルはUTF-8形式で保存してください
- 大きな辞書ファイル（1万語以上）を使用すると、起動時の読み込みに時間がかかる場合があります
