## 指摘事項

### 🟡 擬似ツール呼び出しの除去が、引数内の丸括弧で壊れる

`chat-ai-service.ts` の擬似ツール呼び出しを除去する正規表現は、最初に出現する `)` でマッチを終えます。そのため、引数値自体に丸括弧が含まれると、擬似呼び出しの後半が応答文に残ります。

例:

```text
[manageTasks(action="add", title="定例会議(営業)")]
```

上記では `title` の値に含まれる最初の `)` までしか除去できず、残りの `")]` がユーザーに表示されます。タスク名や予定名で丸括弧を使うことは自然なため、発生し得るケースです。

**推奨対応**:

- 角括弧・引用符・丸括弧のネストを扱える走査処理へ置き換える。
- 少なくとも `title="会議(定例)"` を含む擬似ツール呼び出しを除去できるユニットテストを追加する。
