# このアプリについて

このアプリは、エージェント機能を備えたチャットアプリです
一般のチャットアプリとは違い、相手は AI Agent となります。

この AI Agent(以下、マスコット) には下記の特徴があります

## マスコットの特徴

* 全身や、バストアップ描画あり
* マスコットを複数変更できる
* マスコットには、表情は服装を変更できる
* マスコットの描画は 2D で行う
* チャットへの返答として、自動的に表情を変更する
* 服装は、主にプレイヤーが指定した時変更する
* Agent Skill を作る機能有り
* 場合によっては、複数体のキャラクターと話す場合がある
* キャラクター通しで話す場合もある
* マスコットの話す内容は、LLMによって生成する
* マスコットの話す内容は、音声生成AIが生成して再生する（音声生成AIが利用可能な場合）
* マスコットの表情はアニメーションする（アニメーション設定がある場合）
* マスコットのポーズはアニメーションする（アニメーション設定がある場合）

# 画面について

## メイン画面

* メイン画面には、マスコットと、チャットウィンドウが表示される
* マスコットの背景は透明
* チャットウィンドウの背景は半透明
* マスコットをD＆Dで移動できる。チャットウィンドウも追従する
* チャットウィンドウをD&Dすると、マスコットも追従する
* マスコットを右クリックで設定画面を表示する
* チャットウィンドウはスクロール可能で、最新のメッセージは最下に追加される
* チャットウィンドウからはみ出した部分のチャットは表示しない
* チャットウィンドウのヘッダー部分にはツールバーが表示する
    * 「＋」ボタン：新規チャットを作成
    * 「History Icon」ボタン：チャット履歴を表示して選択
* チャットウィンドウのフッター部分にはメッセージ送信用のコントロールがある

## マスコット設定画面

設定画面は下記の機能を持ちます

* マスコット一覧パネル
* Chat AI 設定パネル
* 音声 AI 設定パネル
* 画像生成 AI 設定パネル
* 動画生成 AI 設定パネル
* API KEY 設定パネル

## マスコット一覧パネル

表示するマスコットを指定・追加するパネル
各マスコットのカバー画像と名前を一覧表示するコントロール、マスコットの設定情報を表示するコントロールがある
一覧からマスコットを選択すると、メイン画面に表示されるマスコットが切り替わり、マスコットの設定表示コントロールも切り替わる
マスコットの設定情報コントロールには下記がある

* 名前
* 服装一覧
* 表情一覧
* 性格などをの簡単な紹介文書
* Chat AI Engine と Model の値
* 音声 AI Engine と Model の値
* 編集ボタン → マスコット編集画面に遷移する

### マスコット編集画面

マスコットの設定を編集する画面です。
マスコット設定には下記があります

* マスコット名
* プロフィール設定
  キャラクターのプロフィールを設定する
  この設定が、キャラクターのシステムプロンプトとして使われる
  * プロフィール編集画面
* カバー画像
  マスコット一覧で使われる画像
* 服装一覧
  服を着たキャラクターの画像が一覧で表示される
  服装は追加できる。プロンプトで生成するか、パソコンの画像をアップロードする
  各服装には色々な角度からの画像を表示するボタンがある。色々な角度からの画像を生成するボタンがある
* 表情一覧
  キャラクターの表情画像一覧
  表情は追加できる。プロンプトで生成するか、パソコンの画像をアップロードする
  各表情にはアニメーション再生ボタンと、生成ボタンがある
* ポーズ一覧
  キャラクターのポーズ画像一覧
  ポーズは追加できる。プロンプトで生成するか、パソコンの画像をアップロードする
* Chat AI Engine と Model の値
* 音声 AI Engine と Model の値


## Chat AI 設定パネル

* Chat AI エンジンの種類の一覧
* 各 Chat AIエンジン のモデル一覧
* Chat AI エンジンを選択すると、モデル一覧が更新される
* 各モデルの設定値（コンテキスト長など）の設定コントロール

## 音声 AI 設定パネル

* 各 音声生成 AIエンジン のモデル一覧
* 音声生成 AI エンジンを選択すると、モデル一覧が更新される
* 各モデルの設定値の設定コントロール

## 画像生成 AI 設定パネル

* 画像生成 AI エンジンの種類の一覧
* 各 画像生成 AIエンジン のモデル一覧
* 画像生成 AI エンジンを選択すると、モデル一覧が更新される
* 各モデルの設定値の設定コントロール

## 動画生成 AI 設定パネル

* 動画生成 AI エンジンの種類の一覧
* 各 動画生成 AIエンジン のモデル一覧
* 動画生成 AI エンジンを選択すると、モデル一覧が更新される
* 各モデルの設定値の設定コントロール

## API Key 設定パネル

* AIエンジン毎の API KEY 設定パネル
