UIレイアウト実装指示書 (Chat AI Settings)
1. 全体構造 (Layout Overview)

画面全体は、モダンなデスクトップアプリ（Windows 11スタイル）のデザイントレンドに準拠し、角丸のカード、適切なインセット、柔らかいドロップシャドウを配置してください。
画面は左から順に3つの垂直カラム（3列構造）で構成します。

    Window Size: デフォルト幅 1024px × 高さ 768px（レスポンシブ対応）

    Theme: ライトモード（背景は非常に薄いグレー ##F3F4F6 または微細なグラデーション）、アクセントカラーは「ソフトブルー（爽やかな青）」。

2. カラム別の詳細仕様 (Column Specifications)
カラム1: メインナビゲーション（左端・狭幅）

1枚目・2枚目のデザインを継承したアプリ全体のサイドメニューです。

    幅: 全体の約 20%

    背景: 透明、または非常に薄いグレー

    配置項目:

        Mascot

        Chat AI (★現在選択状態：背景にソフトブルーの角丸ハイライト、文字色を強調)

        Voice AI

        Image AI

        Movie AI

        API Keys

    スタイル: 各項目は十分なパディング（上下 12px）を持ち、ホバー時に薄いグレーのフィードバックを表示。

カラム2: AI Engine 選択リスト（中央・中幅）

新しくデザインしたプロバイダー選択エリアです。ドロップダウンではなく固定の縦型リスト（ListView）として実装してください。

    幅: 全体の約 40%

    背景: 白（#FFFFFF）、角丸（BorderRadius: 8px）、薄い外枠（Border: 1px solid #E5E7EB）。

    ヘッダー:

        上部にタイトル「Chat AI Settings」（大文字・太字）を配置。

        その下に「AI Engine (Select Provider)」というエリア見出しを配置。

    リストアイテム（以下の6項目を縦に並べる。各項目はアイコン＋名称＋説明文の構成）：

        Gemini (★現在選択状態：背景をライトブルーにし、右端にチェックマークを表示)

        Claude

            アイコン: Claudeの公式アイコンを利用する https://ja.wikipedia.org/wiki/%E3%83%95%E3%82%A1%E3%82%A4%E3%83%AB:Claude-ai-icon.svg

カラム3: モデル・パラメータ設定（右端・広幅）

中央のカラム2で選択されたエンジン（現在はGemini）に紐づく、詳細設定を行うエリアです。

    幅: 全体の約 40%

    背景: 白（#FFFFFF）、角丸（BorderRadius: 8px）、薄い外枠。

    配置項目（上から順に垂直スタック）：

        エリアタイトル: Model and Parameters（太字）

        Model（ドロップダウン）:

            ラベル: Model:

            コンボボックス: Gemini 3-Flash Preview [思考] (右端に下矢印)

            ボタン: コンボボックスの右隣に Refresh ボタン（角丸、薄いグレー背景）を配置。

        Temperature（スライダー）:

            ラベル: Temperature (左側) / 現在値: 1.5 (右側)

            コントロール: 横型シークバー。つまみと通過済みのバーはアクセントブルー。

        Top-K（スライダー）:

            ラベル: Top-K (左側) / 現在値: 20 (右側)

            コントロール: 横型シークバー。

        トグルスイッチ群 (3つ):

            Top-K Toggle ── [ スイッチ：OFF (グレー) ]

            Audio Random Elititors ── [ スイッチ：OFF (グレー) ]

            Bridges Ropherly Models ── [ スイッチ：ON (ブルー) ]

        チェックボックス:

            [✓] Filter Chat-only Models (チェック時はブルー)

3. フッターボタン (Footer Actions)

右下のエリア（カラム3の最下部、または画面全体の右下）に、設定を確定・破棄するためのアクションボタンを配置します。

    配置: 右詰め、横並び（Row layouts）

    Cancel ボタン:

        アイコン: ✕（赤い✕マーク）

        テキスト: Cancel

        スタイル: 背景は白または薄いグレー、文字とアイコンはダークグレー。

    Save ボタン:

        アイコン: 💾（フロッピー/保存マーク）

        テキスト: Save

        スタイル: 背景は薄いブルーまたはグレー、文字とアイコンはダークブルー。

4. スタイリング・実装の注意点 (Coding Notes)

    余白 (Padding/Margin): 各カラムの間隔は 16px から 20px のマージンを空け、窮屈にならないようにしてください。

    フォント: 視認性の高いサンセリフ体（Segoe UI、Arial、Noto Sans等）を使用し、文字のジャンプ率（強弱）を明確にしてください。

    コンポーネント化: カラム2の「リストアイテム」は、アイコン・タイトル・説明文を引数に取る再利用可能なコンポーネントとして定義してください。
