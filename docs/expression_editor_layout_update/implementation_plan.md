# 表情エディタの大画面化・縦スリムリストレイアウト変更の実装計画

表情エディタのUIデザインを大幅に強化し、操作性と視認性を向上させます。具体的には、表情スロットを従来のグリッド表示から「左側のスクロール可能な縦リスト（幅240px）」に変更し、右側の表情位置調整プレビュー領域を大幅に拡大します。また、画像クロップUIの導入、感情割り当て画面の分離、およびウィンドウの座標・サイズ永続化機能を実装します。

## ユーザー確認事項

> [!NOTE]
> 表情エディタ内での位置調整や拡大率は、マスコットウィンドウに即時反映されるのではなく、表情エディタを閉じたタイミング（または「保存」ボタン押下時）に一括して同期・永続化されます。これによりI/O負荷を軽減し、快適な操作感を実現します。

## 提案する変更内容

### [Component: Settings Window UI]
表情エディタおよび感情割り当て画面のレイアウトを一新します。

#### [MODIFY] [SettingsWindow.vue](file:///c:/workspace/workspace-win/DesktopAiMascot/src/components/settings/SettingsWindow.vue)
- **SillyTavern 28感情互ロットの保証**:
    - 感情名（「通常」, 「喜び」, 「怒り」など）を28個の日本語感情として定義。
    - `ensure28Expressions` 関数により、アセットが存在しない場合もデフォルトの28スロットが空枠として表示されるようにします。
- **大画面表情エディタモーダルの実装**:
    - **左カラム (幅240px)**: 28感情が縦並びにスクロール表示される「縦スリムリスト」を配置。各感情のサムネイルと、画像未設定時の点線枠プレースホルダーを表示。
    - **右カラム (プレビュー＆コントロール)**:
        - プレビュー領域を高さ520px、プレビューサイズ420pxに超大型化。
        - 縦スライダー（Y軸調整、高さ340px）をプレビューの右側に垂直配置。
        - 横スライダー（X軸調整）とスケールスライダーをプレビュー下部に配置.
        - クロップ呼び出し用の「切り抜き直す」「画像から切り出し」ボタンを追加。
        - 標準（通常表示）表情に設定するためのチェックボックスを追加。
- **画像クロップ (Crop) UI モーダルの追加**:
    - 画像を読み込み、Canvas上でドラッグ＆ドロップによって切り抜き範囲（正方形）を指定し、Base64として表情スロットに保存するトリミング機能を実装。
- **感情割り当てモーダル (isAssigningEmotionsModal) の分離**:
    - AIスプライトインポート実行完了後、一時トレイを表示するのではなく、独立した感情割り当て画面を開きます。
    - 上部にインポートされたスプライトトレイを配置し、下部に28感情スロットを配置。スプライトを選択してスロットをクリック（またはドラッグ）することで視覚的にマッピングできます。

### [Component: Main Process Window States]
設定ダイアログの起動位置とサイズをアプリ終了後も引き継ぐための永続化を行います。

#### [MODIFY] [main.ts](file:///c:/workspace/workspace-win/DesktopAiMascot/electron/main.ts)
- `settingsWindow` のクローズ時に `win.getBounds()` で座標とサイズ（X, Y, Width, Height）を取得し、`config.json` に保存します。
- 次回の設定画面起動時に、保存された座標とサイズを復元して `BrowserWindow` を作成します。

## 開発・検証計画

### 自動コンパイル & ビルド検証
- `cmd /c npm run build` を実行して、TypeScript コンパイルエラーおよび Vite のプロダクションビルドが一切警告なく 100% 成功することを確認します。

### 手動検証項目
1. **設定画面の永続化**:
    - 設定画面を起動し、サイズや位置を変更して閉じた後、再度開いたときに同じサイズと位置で復元されることを確認します。
2. **表情エディタモーダルの検証**:
    - 表情アセットタブから「大画面エディタ」を開き、左側に28感情の縦スリムリストがスクロール表示され、右側に特大プレビューと縦・横スライダーが表示されることを確認します。
    - 縦スライダーを動かしたときに、プレビュー画像がリアルタイムに上下に動くことを確認します。
    - リセットボタン、表情解除ボタンが正常に動作することを確認します。
3. **クロップ機能の検証**:
    - 「画像から切り出し」をクリックしてローカル画像を選択し、正方形のクロップ枠を動かして「切り抜き実行」した結果がスロットに適用されることを確認します。
4. **AIスプライト感情割り当ての検証**:
    - AIスプライトインポートを実行し、自動的に感情割り当てモーダルが開くこと、およびトレイから感情スロットへの割り当てが正常に動作することを確認します。
