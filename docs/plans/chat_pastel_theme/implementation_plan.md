# チャットウィンドウの淡いパステル調（ライト・グラスモーフィズム）への変更計画

マスコットのチャットウィンドウ（`ChatPanel.vue`）のデザインを、従来のダーク調のグラスモーフィズムから、設定画面の白基調に調和する、非常に美しく可愛らしい「淡いパステル調のライト・グラスモーフィズム」へ変更します。
すりガラス効果（`backdrop-filter`）を活かしつつ、淡いラベンダーやパープルのパステルカラーを用いて意匠性を向上します。

## ユーザーレビュー要求事項

> [!NOTE]
> ルート要素から `app-dark` クラスを除去（または `app-light` へ変更）し、全体を淡いパステル調に調和させます。

## オープンクエスチョン

特になし。

---

## 提案される変更

### フロントエンド（Vue.js / CSS）

#### [MODIFY] [ChatPanel.vue](file:///c:/workspace/workspace-win/DesktopAiMascot/src/components/ChatPanel.vue)
- テンプレートのルート要素 `<div class="chat-wrapper app-dark">` から `app-dark` クラスを除去します。
- スタイルブロック（CSS）の配色を以下の「淡いパステル調のライト・グラスモーフィズム」に全面的に刷新します：
    - **全体背景 (`chat-wrapper`)**:
        - 明るい半透明の純白（`background: rgba(255, 255, 255, 0.65)`）とすりガラス効果（`backdrop-filter: blur(20px)`）を組み合わせます。
        - 境界線を `border: 1px solid rgba(255, 255, 255, 0.4)` とし、デスクトップ上で美しく浮かび上がる淡い影（`box-shadow: 0 8px 32px rgba(0, 0, 0, 0.06)`）を適用します。
    - **ヘッダー (`chat-header`)**:
        - 半透明白（`background: rgba(255, 255, 255, 0.3)`）とし、下境界線を `rgba(0, 0, 0, 0.05)` に設定します。
        - タイトル文字色をダークスレートグレー（`#475569`）にし、ボタン色を `color: #64748b`（ホバー時 `#0f172a`, 丸背景 `#000000` の極淡透明度）にします。
    - **フキダシ部分 (`bubble`)**:
        - **マスコット側のフキダシ**: 非常に淡い半透明のパステルラベンダー（`background: rgba(243, 232, 255, 0.7)`, 境界線 `1px solid rgba(168, 85, 247, 0.1)`）を適用し、文字色は深紫（`#4a2c7a`）で可読性を極限まで高めます。
        - **ユーザー側のフキダシ**: 明るくやさしいパステルパープル（`background: #e9d5ff`）を適用し、文字色は深めの紫（`#581c87`）にします。
    - **メッセージ入力エリア (`chat-footer` / `message-input`)**:
        - 入力背景を半透明白（`background: rgba(255, 255, 255, 0.5)`）とし、境界線を `1px solid rgba(0, 0, 0, 0.08)` とします。フォーカス時は白背景＋パープルボーダー（`#a855f7`）に変化させます。
        - 送信ボタンはパステルパープル（`background: #c084fc`, ホバー時 `#a855f7`）に設定します。
    - **スクロールバーのカスタマイズ**:
        - スクロールバーのつまみを `background: rgba(0, 0, 0, 0.08)` に変更し、ライトテーマ下でも自然に見えるように調整します。

## 検証計画

### 修正内容のビルド検証
- `cmd.exe /c npm run build` を実行し、CSSルール変更後にビルドエラーが発生しないことを確認します。

### 手動検証
- VS Codeからアプリケーションを起動し、チャットウィンドウを表示します。
- チャットウィンドウ全体が、すりガラス効果を伴う美しい「淡いパステル調」で可読性高く表示されていることを確認します。
- メッセージを送信し、マスコット側のラベンダー調フキダシと、ユーザー側のパステルパープルフキダシが美しく交互に並ぶことを確認します。
- メッセージ入力時のフォーカスアニメーションや、送信ボタンのパステルなグラデーション/ホバーが滑らかに機能していることを確認します。
