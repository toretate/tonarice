# 実装計画: 設定画面（SettingsWindow）のコンポーネント分割・ファイルリファクタリング ＆ 表情エディタの白基調（ライトモード）化

## 目的
現在 2,500行を超え、複数のモーダルと複雑な設定ロジックが混在して肥大化している `SettingsWindow.vue` を、プロジェクト開発ルール（「１ファイルに機能を詰め込まない」）に従い、機能ごとに別ファイル（コンポーネント）へ適切に分割・リファクタリングします。

同時に、現在ダークモード基調になっている表情エディタモーダルを、メインの設定画面と同様に**白・ライトカラーを基調とした、清潔感と透明感のあるプレミアムなライトモードデザイン**に刷新します。

---

## ユーザー確認事項
リファクタリングおよびデザイン刷新にあたり、以下の動作確認や仕様に関する懸念点をご確認ください。

> [!IMPORTANT]
> **状態管理の整合性の維持**
> 分割後も設定のリアルタイム保存（`saveSettings`）や、親コンポーネントとの双方向バインディング（`v-model`）が正しく動作するように、Vue 3の `defineProps`, `defineEmits` を用いて、データの整合性を確実に保護します。
>
> **表情エディタの白基調（ライトモード）デザイン刷新**
> 単に背景を白くするだけでなく、透明な背景を持つ表情アセット（イラスト）が白飛びして見えなくならないよう、プレビューエリアには上品な「薄いグレーの市松模様（チェッカーボード）パターン」を適用し、透過イラストが綺麗に際立つプロ仕様のプレミアムデザインを実装します。

## 提案するファイル構成（設計）

`src/components/settings/` の下に以下の5つのコンポーネントに整理します。

### 1. `SettingsWindow.vue` [MODIFY]
- **役割**: 環境設定画面全体のベースレイアウト、サイドバーナビゲーション、および一般環境設定パネル（AI、音声、ウィンドウ、APIキーなど）のホスト。
- **変更内容**: マスコット詳細設定や各種モーダルの記述を削除し、代わりに `MascotSettings.vue` をインポートして呼び出します。

### 2. `MascotSettings.vue` [NEW]
- **役割**: マスコット一覧の表示、プロフィールの編集、全身像（立ち絵）アセット管理、表情アセットグリッド（28スロット表示）の表示。
- **内包するモーダル**:
  - `ExpressionEditorModal` (表情大画面エディタ)
  - `EmotionAssignmentModal` (感情割り当て)
  - `ImageCropModal` (トリミング)
- **依存する状態**: `mascots`, `activeMascotId`, `editingMascot`

### 3. `ExpressionEditorModal.vue` [NEW]
- **役割**: 表情編集と位置調整の大画面エディタモーダル（L1535〜L1740付近）。
- **デザイン変更**: **白基調のライトモードデザインに刷新。**
- **Props**:
  - `visible`: 表示状態
  - `editingMascot`: 編集対象マスコットデータ
- **Emits**:
  - `close`: 閉じる
  - `save`: 編集データの反映
  - `crop-current`: 現在の表情をトリミング
  - `crop-new`: ローカル画像からトリミング

### 4. `EmotionAssignmentModal.vue` [NEW]
- **役割**: AIスプライトインポート時の感情割り当てモーダル（L1743〜L1818付近）。
- **デザイン変更**: 白基調に合わせたライトテーマUI。
- **Props**:
  - `visible`: 表示状態
  - `editingMascot`: 編集対象マスコット
  - `scannedSprites`: 検出済みスプライト一覧
- **Emits**:
  - `close`: 閉じる
  - `assign`: スプライト割り当て実行
  - `update:scannedSprites`: スプライト一覧更新

### 5. `ImageCropModal.vue` [NEW]
- **役割**: 表情画像のクロップ（トリミング）モーダル（L1821〜L1890付近）。
- **デザイン変更**: 白基調に合わせた使いやすいクロップUI。
- **Props**:
  - `visible`: 表示状態
  - `imageSrc`: トリミング対象画像（パスまたはBase64）
- **Emits**:
  - `close`: 閉じる
  - `crop`: クロップ成功（Base64のクロップ画像を引数として渡す）

## 白基調（ライトモード）配色設計

表情エディタの各要素を以下のようにライトモード用の高品質な配色に変更します。

| 要素 | 旧カラー (ダークモード) | 新カラー (白・ライトモード) | 意図 |
| :--- | :--- | :--- | :--- |
| **モーダル外枠** | `#1e293b` (ダークSlate) | `#ffffff` (純白) | 清潔感のあるメインカラー |
| **モーダル背景** | `rgba(15, 23, 42, 0.85)` | `rgba(241, 245, 249, 0.85)` + ぼかし | 透明感のあるライトグレーのオーバーレイ |
| **文字色（大）** | `#f8fafc` (白) | `#1e293b` (深いグレー) | 高いコントラストで読みやすいタイポグラフィ |
| **文字色（小）** | `#94a3b8` (薄いグレー) | `#475569` / `#64748b` | 補助テキストの上品な明度 |
| **境界線** | `rgba(255, 255, 255, 0.1)` | `rgba(0, 0, 0, 0.08)` / `#e2e8f0` | 薄くすっきりとした仕切り線 |
| **プレビューカード背景**| `#020617` (ブラック) | 薄いグレーの市松模様 (チェッカー柄) | **白いアバター画像・透過アセットの白飛び防止** |
| **スロットアイテム背景**| `rgba(255, 255, 255, 0.02)` | `#f8fafc` (超極薄グレー) | リストの個別アイテムをカード化 |
| **スロット（アクティブ）**| 紫色バックライトトーン | 薄い紫背景 (`#f5f3ff`) + 紫境界線 (`#a855f7`) | アクティブ状態を鮮やかに強調 |
| **コントロールパネル背景**| `#0f172a` (ダークネイビー) | `#f8fafc` (極薄グレー) + 薄い境界線 | 調整エリアを分かりやすくセクション化 |
