# 設定画面の白基調（ライトモード）への変更計画

設定画面のデザインシステムを黒基調のダークモードから、明るく美しく洗練された白基調（ライトモード）へと変更します。
単に白くするだけでなく、上品なオフホワイト、ソフトな境界線、グラスモーフィズム調の影を用いたプレミアムで高級感のあるライトテーマを構築します。

## ユーザーレビュー要求事項

> [!NOTE]
> 設定画面のルート要素のクラスを `app-dark` から除去し、PrimeVue が標準の美しいライトモードでレンダリングされるように調整します。

## オープンクエスチョン

特になし。

## 提案される変更

### フロントエンド（Vue.js / CSS）

#### [MODIFY] [SettingsWindow.vue](file:///c:/workspace/workspace-win/DesktopAiMascot/src/components/settings/SettingsWindow.vue)
- テンプレート内のルート要素 `<div class="settings-layout app-dark">` を、ダークモードを解除するために `<div class="settings-layout">`（または `app-light` クラス）へ変更します。
- スタイルブロック（CSS）の配色を以下の「洗練されたプレミアム・ライトテーマ」に全面的にリファクタリングします：
    - **全体背景**: `#f8fafc` (微小なブルーグレー調の明るいスレートカラー)
    - **左サイドバー背景**: `#ffffff` (境界線 `border-right: 1px solid rgba(0, 0, 0, 0.06)`)
    - **メインコンテンツ背景**: `#f1f5f9` (淡いグレー)
    - **プレミアムカード背景**: `#ffffff` (境界線 `border: 1px solid rgba(0, 0, 0, 0.04)`, 影 `box-shadow: 0 4px 20px rgba(0, 0, 0, 0.04)`)
    - **テキストカラー**:
        - メイン文字色: `#1e293b` (ダークスレートグレー)
        - ラベル・説明文字色: `#64748b` (ソフトなクールグレー)
    - **メニューアイテム**:
        - 未選択ホバー時: `color: #0f172a`, `background: rgba(0, 0, 0, 0.02)`
        - 選択（アクティブ）時: パープルアクセント（`color: #a855f7`, `background: rgba(168, 85, 247, 0.06)`）
    - **マスコットアイテム選択**:
        - 未選択時: `background: #ffffff`, `border: 1px solid rgba(0, 0, 0, 0.06)`
        - 選択時: `border-color: #a855f7`, `background: rgba(168, 85, 247, 0.04)`
    - **接続テストステータス枠**:
        - 接続成功時: `background: rgba(34, 197, 94, 0.04)`, `border: 1px solid rgba(34, 197, 94, 0.15)`, `color: #15803d` (やさしい緑)
        - 接続失敗時: `background: rgba(239, 68, 68, 0.04)`, `border: 1px solid rgba(239, 68, 68, 0.15)`, `color: #b91c1c` (やさしい赤)
        - 未確認時: `background: rgba(0, 0, 0, 0.02)`, `border: 1px solid rgba(0, 0, 0, 0.05)`, `color: #64748b`

## 検証計画

### 修正内容のビルド検証
- `cmd.exe /c npm run build` を実行し、CSSルール変更後にエラーが発生しないことを確認します。

### 手動検証
- 設定画面を起動し、全体が美しく洗練された白基調（ライトモード）で崩れなく表示されていることを確認します。
- 各種入力欄（InputText, Select, Password等）、ボタン、カードがライトモード下で可読性高く表示されていることを検証します。
- メニューのホバー・アクティブ時の表示が美しく機能していることを確認します。
- 接続確認時のグリーン調／レッド調のステータス表示が、白基調の背景下でも視覚的に美しく調和していることを確認します。
