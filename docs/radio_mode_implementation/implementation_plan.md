# ラジオモードの実装計画

ラジオモード（マスコットがラジオパーソナリティとして1人でトークを繰り広げるモード）をチャット画面に組み込むための実装計画です。

---

## ユーザー確認が必要な事項

> [!NOTE]
> - **ラジオモードON時のシステムプロンプトの調整**: 
>   ラジオモードがONの時には、マスコットがリスナーに向けて話しかけるラジオDJのような口調になるよう、システムプロンプトに「ラジオパーソナリティ風 of 振る舞い指示」を動的に追加します。
> - **チャットテキストの判定フレーズ**:
>   「ラジオモード」という言葉が含まれ、「オン」「ON」「開始」「起動」などの肯定フレーズがあればONにし、「オフ」「OFF」「終了」「停止」などの否定フレーズがあればOFFにする簡易的なルールベース判定を組み込みます。

---

## 提案する変更内容

### 1. マスコットの状態管理 (Pinia ストア)
#### [MODIFY] [mascot.ts](file:///c:/workspace/workspace-win/DesktopAiMascot/ui/src/store/mascot.ts)
- `isRadioMode` 状態（デフォルト `false`）を定義します。
- ラジオモードを切り替える `setRadioMode` アクションを追加します。

### 2. チャット画面UI (Vue コンポーネント)
#### [MODIFY] [ChatPanel.vue](file:///c:/workspace/workspace-win/DesktopAiMascot/ui/src/components/ChatPanel.vue)
- ヘッダーのアクション部分にラジオモード切り替え用アイコンボタン（マイクアイコン：`pi-microphone` を想定）を追加します。
- ラジオモードが有効な間、ボタンのカラーを警告色/アクティブ色（例: 紫や赤のグラデーション、あるいはパルスアニメーション）にして「ON状態」であることが視覚的にわかるようにします。
- ボタンクリックで `mascotStore.setRadioMode(!isRadioMode)` を呼び出し、トグルできるようにします。

### 3. チャット通信制御 (WebSocket / API連携)
#### [MODIFY] [useChatConnection.ts](file:///c:/workspace/workspace-win/DesktopAiMascot/ui/src/components/chatpanel/useChatConnection.ts)
- `sendMessage` メソッドにて、ユーザーの入力内容（`userQuery`）をパースし、「ラジオモード」をON/OFFするチャット指示を検出する処理を追加します。
  - **ON指示例**: 「ラジオモードを開始して」「ラジオモードON」など
  - **OFF指示例**: 「ラジオモードを終了して」「ラジオモードOFF」など
- `sendMessage` 内でAPI/WebSocketへ送信する `systemPrompt` に、ラジオモードがONである場合の専用の指示文を自動的に追加します。
  - **追加するプロンプト指示（例）**:
    ```
    あなたは現在、1人喋りの「ラジオパーソナリティ（MC）」としてラジオ番組を配信しています。
    親しみやすい口調で、目の前のリスナー（マスター）に向けてラジオ風の楽しいトークを展開してください。
    挨拶（「リスナーのみなさんこんにちは！」「お便りありがとうございます」など）や、ラジオらしい言い回しを効果的に使ってください。
    ```

## 動作確認・検証計画

### 手動テスト
1. **アイコンクリックによる切り替えテスト**:
   - チャットパネル of ヘッダーにあるマイクアイコンをクリックしたときに、アイコンがアクティブ状態（カラー変化など）になり、ストアの `isRadioMode` がトグルすることを確認します。
2. **チャット指示による自動切り替えテスト**:
   - 入力欄に「ラジオモードを開始して」と入力して送信した際、ヘッダーのマイクアイコンが自動的にオン（アクティブ）に切り替わることを確認します。
   - 「ラジオモードを終了して」と入力して送信した際、マイクアイコンがオフに切り替わることを確認します。
3. **ラジオトークの動作確認**:
   - ラジオモードONの状態で適当なテキストを送信した際、マスコットのセリフが「みなさんこんにちは！」やラジオDJ風の口調（一人語りのラジオ番組風）に変化することを確認します。
