# タスク: チャットウィンドウの背景設定追加

## 概要
チャットウィンドウの背景をより柔軟にカスタマイズできるよう、ウィンドウ設定に背景設定を追加します。背景色（既存）に加えて、背景画像の設定（画像の選択、不透明度、フィット方法）を行えるようにし、チャットウィンドウへリアルタイムに適用します。

## 要件
1. **設定値の追加** (`ui/src/store/config.ts`):
   - `chatBackgroundImage`: 背景画像のパス（ローカル画像選択によるBase64データまたはURL）
   - `chatBackgroundImageOpacity`: 背景画像の不透明度 (0.0 〜 1.0)
   - `chatBackgroundImageFit`: 背景画像のフィット方法 ('cover' | 'contain' | 'fill' | 'tile')

2. **設定画面のUI追加** (`ui/src/components/settings/WindowSettingsPanel.vue`):
   - 背景画像の選択ボタン（既存 of `selectLocalImage` を使用）
   - 背景画像のクリアボタン
   - 背景画像の不透明度調整スライダー (0% 〜 100%)
   - 背景画像のフィット方法選択ドロップダウン ('カバー', '全体表示', '引き伸ばし', '並べて表示')
   - **左右 2Pane へのレイアウト変更**:
     - 左側ペイン: 各種設定コントロール
     - 右側ペイン: 設定値がリアルタイムに反映されるチャットプレビュー領域

3. **チャットウィンドウへの反映** (`ui/src/components/ChatPanel.vue`):
   - チャットウィンドウの背景に、設定された背景画像と不透明度、フィット方法を適用する。
   - テキストの読みやすさを損なわないよう、背景画像レイヤーを背面に配置し、不透明度を反映させる。
