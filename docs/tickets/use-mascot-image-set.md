# 配列で持っている画像セットを MascotImageSet を使うように修正する

## 概要

画像セットを `string[]` や画像パス配列ベースで扱っている箇所を見直し、`MascotModel` と `MascotImageSet` を 1:1 で扱う構成に寄せる。

## 目的

- `MascotModel` が単一の `MascotImageSet` を保持する
- チャット画面の既定表示は `front` を使う
- 感情差分と方向画像は `MascotImageSet` から明示的に参照する
- 画像名から方向や感情を分類する処理を共通化する
- `cover.png` は代表画像として扱い、チャット用セットからは除外する

## 完了条件

- 画像パス配列への直接依存を削減できている
- 表示側が `front` ベースで画像を選択している
- `MascotImageSet` の構築ロジックに対するテストがある
- 関連コードのビルドと対象テストが通る
