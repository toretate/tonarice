# チャットウィンドウと設定ダイアログの表示トリガーを変更する

## 概要

マスコットを操作する際のチャットウィンドウと設定ダイアログの表示トリガーを変更し、より直感的に操作できるようにします。

## 目的

- チャットウィンドウはマスコットを左クリック（シングルクリック）で表示・非表示をトグルできるようにする。
- 設定ダイアログはマスコットを右クリックで表示できるようにする。
- ドラッグ操作によるマスコットウィンドウの移動と、クリック操作の判定を両立させる。

## 完了条件

- マスコット部分を左クリックした際に、チャットウィンドウの表示・非表示がトグルされること。
- マスコット部分を右クリックした際に、設定ダイアログが表示されること。
- マスコット部分をドラッグしてウィンドウを移動させることができること。また、ドラッグ終了時にはクリック判定（チャットウィンドウ表示）が行われないこと。
- 関連コードのビルドおよび動作に問題がないこと。
