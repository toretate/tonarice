# Pinia（状態管理ライブラリ）の導入とセットアップ

## 概要

アプリケーションのグローバル状態管理をシンプルかつ型安全に行うため、Vue 3 の推奨状態管理ライブラリである `Pinia` を導入し、基本的なストアの設定を行います。

## 目的

- 複数コンポーネント間で共有される状態（設定値、チャット表示状態など）を一元管理できるようにする。
- Vue 3 + TypeScript に適した型安全な状態管理を導入し、コードの見通しと保守性を高める。
- グローバル状態管理を簡易な reactive オブジェクトやイベントバスから Pinia ストアへ移行する基盤を作る。

## 完了条件

- `package.json` に `pinia` が依存関係として追加されていること。
- `src/main.ts` で Pinia がアプリケーションに登録されていること。
- 動作確認用のサンプルストア（または動作確認コード）が用意されており、テストおよびビルドが正常に通過すること。
