# DesktopAiMascot UI/UX デザイン指針 (DESIGN GUIDELINES)

本ドキュメントは、DesktopAiMascot アプリケーションにおける統一感のある美しくモダンな UI/UX を構築・維持するためのデザイン指針です。
新規画面の追加や既存 UI の改修時には、本ガイドラインに沿って実装を行ってください。

---

## 1. デザインコンセプト (Design Concept)

### 「クリーン・モダン・フレンドリー」
*   **クリーン＆モダン**: 無駄な装飾を省き、余白（ホワイトスペース）とソフトな影を活用した Fluent / Material デザイン調のスタイルを採用します。
*   **フレンドリー**: マスコットキャラクターを引き立てるため、淡いブルーグレーとクリーンなホワイトを基調とし、親しみやすくストレスのない操作感を提供します。

## 3. タイポグラフィ (Typography)

フォントサイズおよびスタイルは、以下のように体系化して適用します。

*   **大見出し (HeaderLarge - ページタイトル)**
    *   フォントサイズ: `24px`
    *   用途: 設定画面などの各ページの最上部タイトル。
*   **中見出し (HeaderMedium - セクションタイトル)**
    *   フォントサイズ: `18px`
    *   用途: パネルやカード内のグループ名。
*   **小見出し (HeaderSmall - コントロール見出し)**
    *   フォントサイズ: `16px`
    *   用途: 各項目の細分化されたグループタイトル。
*   **標準テキスト (Label/Value)**
    *   フォントサイズ: `14px`
    *   用途: パラメータ名、スライダーの現在値、通常のテキスト表記。

## 4. レイアウトとスペース (Layout & Spacing)

UI が窮屈に見えたり、逆に散漫に見えたりするのを防ぐため、コントロール間のスペース（Separation）やマージンは原則として以下の数値に固定します。

### 4.1. マージン (Margins)
*   **ページ全体の外周余白 (MarginContainer)**: 一律 `12px` 〜 `16px` (基本は `16px` を推奨)
*   **パネル/カードの内側余白 (MarginContainer)**:
    *   標準的な設定パネル: `16px`
    *   情報を集約するインフォパネル: `20px`

### 4.2. コントロール間の隙間 (Separation)
*   **大カテゴリ間 (左右のメインパネル同士など)**: `16px` 〜 `20px` (HBoxContainer)
*   **中カテゴリ間 (パネル内の縦並びセクションなど)**: `12px` 〜 `16px` (VBoxContainer)
*   **細かなパラメータ行同士の間隔**: `14px` (VBoxContainer)
*   **パラメータ行内のパーツ同士の間隔 (ラベルと入力欄など)**: `10px` (HBoxContainer)
*   **スライダーとラベルの間隔 (縦並び)**: `4px` 〜 `6px` (VBoxContainer)
*   **フッターのボタン同士の間隔**: `10px` (HBoxContainer)
*   **リスト内のアイテム同士の間隔**: `8px` (VBoxContainer)

### 4.3. 標準ウィンドウサイズ (Window Sizes)
*   **設定画面などの主ダイアログ**: `1150 x 768` (最小サイズ `min_size` にも同値を設定し、リサイズによる極端なレイアウト崩れを防ぐ)

## 5. コントロール別デザイン詳細 (Control Specific Guidelines)

### 5.1. ボタン (Buttons)
ボタンは機能の重要度に応じて角丸やカラーリングを明示的に分けた StyleBoxFlat を適用します。

*   **共通仕様**:
    *   角丸 (Corner Radius): 一律 `8px` (すべての角)
    *   高さ (Height):
        *   通常のフッター/アクションボタン: `36px`
        *   主要な操作ボタン (Edit, Generate 等): `44px` 〜 `48px`
*   **スタイルバリエーション**:
    1.  **プライマリボタン (Primary/Edit)**
        *   背景: ブルー (`#3377f5` / ホバー時 `#2161d9`)
        *   フォント色: 白色 (`#ffffff`)
    2.  **標準/セカンダリボタン (Secondary/Standard/Cancel)**
        *   背景: 白色 (`#ffffff` / ホバー時 `#f5f5f5`) または非常に薄いグレー。
        *   境界線: `1px` 薄いグレー (`#e6e8ed`)
        *   フォント色: ダークグレー (`#333333` / ホバー時 `#1a1a26`)
    3.  **成功/生成ボタン (Creative/Generate)**
        *   背景: 薄緑色 (`#ebfaf0` 相当 / ホバー時 `#d9f5e0`)
        *   境界線: `1px` 緑色 (`#b8e6cc` / ホバー時 `#85d1a6`)
        *   フォント色: 濃い緑色 (`#409959`)
    4.  **危険/削除ボタン (Danger/Remove)**
        *   背景: 極薄いグレー (`#f5f5f5` / ホバー時 `#ececec`)
        *   境界線: `1px` グレー (`#d9d9d9` / ホバー時 `#bfbfbf`)
        *   フォント色: ミディアムグレー (`#666673` / ホバー時 `#333340`)

### 5.2. リスト (List/ItemList)
サイドメニューなどのリストは、選択項目が視覚的に強く認知できるようにします。

*   **サイズ**:
    *   左側のナビゲーションカテゴリリスト: 最小幅 `180px`
    *   マスコット選択などのリスト: 最小幅 `240px`
*   **アイテムの余白 (Content Margin)**:
    *   通常状態: 左 `16px`、上 `8px`、右 `12px`、下 `8px`
    *   選択状態: 左 `12px` (左ボーダーが追加されるため調整)、上 `8px`、右 `12px`、下 `8px`
*   **選択状態のインジケータ**:
    *   選択時背景: 淡いブルー背景 (`#d9ebfa`)
    *   左境界線 (Border Left): 太さ `4px` のブルーボーダー (`#3377f5`) を追加。
    *   角丸: 右上・右下の角丸のみ `8px` とし、選択状態の連続感を演出。

#### 5.2.1. アイコン付きリストアイテム (Icon List Item)
AIプロバイダー選択など、テキストに加えてアイコンを併記するリスト行（PanelContainer / HBox 構成など）では、以下の規則を適用します。

*   **高さ (Height)**: 均一でスリムなリストにするため、1行の固定高（最小高）を一律 **`48px`** とします。
*   **レイアウトと隙間**: 
    *   横並び (`HBoxContainer`) を用い、要素間の間隔 (Separation) は一律 **`12px`** とします。
    *   パディング（Content Margin）は左・右 `12px`、上・下 `6px` とします。
*   **アイコン仕様**:
    *   サイズ: 一律 **`28px` × `28px`**
    *   配置: 垂直方向は中央寄せ (`ShrinkCenter`)、ストレッチモードはアスペクト比維持の中央寄せ (`KeepAspectCentered`)。
*   **テキスト仕様**:
    *   フォントサイズ: **`14px`**
    *   文字色: `#1a1a26`（黒に近いダークグレー）。
*   **右側の追加インジケータ**:
    *   選択されているアイテムには、右端にアクセントブルー（`#3377f5`）のチェックマーク（**`✓`**、サイズ `16px`、垂直中央寄せ）を表示するスタイルを推奨します。
*   **インタラクション (ホバー)**:
    *   マウスホバー時は、アイテム領域全体に対して非常に薄い黒（`rgba(0, 0, 0, 0.05)`）のオーバーレイ角丸背景（角丸 `8px`）を表示し、明確なフィードバックを提供します。


#### 5.2.2. 縦ナビゲーションリスト (VerticalNavigationList)
設定画面などの左側サイドバーで、カテゴリ間の切り替えを行うナビゲーションリストです。詳細は [vertical_navigation.txt](vertical_navigation.txt) 設計仕様書を参照します。

*   **基本仕様**:
    *   コンポーネント名: `VerticalNavigationList`
    *   最小幅: `180px`
    *   1アイテムの高さ: **`34px`** （固定）
    *   レイアウトコンテナ: `VBoxContainer` で管理
*   **コンテナ余白 (Padding)**:
    *   上下: `16px`
    *   左右: `8px`
*   **アイテム余白 (Content Margin)**:
    *   通常状態: 左 `12px`、右 `12px`、上 `0px`、下 `0px`
    *   選択状態: 左 `12px` (左ボーダー `4px` により調整)、右 `12px`、上 `0px`、下 `0px`
*   **3状態のスタイル指定**:
    1.  **Default（未選択）**:
        *   背景: 透明 (`rgba(0, 0, 0, 0.0)`)
        *   フォント色: 中グレー (`#333333`, 80% 不透明度)
        *   角丸: なし
    2.  **Hover（未選択時のホバー）**:
        *   背景: 極薄いグレーのオーバーレイ (`#E4EBF7`, 50% 不透明度)
        *   フォント色: 黒 (`#000000`)
        *   カーソル: `pointer`
        *   角丸: なし
    3.  **Selected（アクティブ）**:
        *   背景: 淡いブルー (`#D2E3FC`, 100% 不透明度)
        *   フォント色: アクセントブルー (`#1A73E8`)
        *   フォント太さ: **Bold**
        *   左境界線: **`4px`** のソリッドボーダー (色: `#1A73E8`)
        *   角丸: 右上・右下のみ `4px`
    4.  **Selected Hover**:
        *   背景: やや濃いブルー (`#C2D9FA`, 100% 不透明度)
        *   フォント色: アクセントブルー (`#1A73E8`)
        *   左境界線: `4px` (色: `#1A73E8`)
        *   角丸: 右上・右下のみ `4px`
    5.  **Pressed（マウスダウン）**:
        *   背景: より濃いブルー (`#B2D1F9`, 100% 不透明度)
        *   フォント色: アクセントブルー (`#1A73E8`)
        *   左境界線: `4px` (色: `#1A73E8`)
        *   角丸: 右上・右下のみ `4px`
*   **アニメーション**:
    *   状態遷移: **`0.2秒`** の滑らかなイージング (`ease`) で背景色と文字色を変更
*   **タイポグラフィ**:
    *   フォントファミリー: システムフォント (`system-ui, -apple-system, sans-serif`)
    *   フォントサイズ: **`14px`**
    *   テキスト配置: 左揃え
    *   テキスト省略: `white-space: nowrap; text-overflow: ellipsis; overflow: hidden;`
*   **キーボード対応**:
    *   Up/Down Arrow キーでのフォーカス移動をサポート
    *   Enter/Space キーでの選択をサポート
    *   Tab キーでのナビゲーションに対応
    *   フォーカス状態: `1px` のダッシュ線フォーカスリング (色: `#1A73E8`) を現在の背景スタイルの上に表示
*   **インタラクション制約**:
    *   全体の対話領域（ヒットボックス）は、アイテム全幅を対象とすること（テキスト領域のみの選択は不可）
    *   テキストが複数行に折り返されることはないようにする（必要に応じて省略）


### 5.3. スライダー (Sliders)
音量、温度 (Temperature) などの調整スライダーは直感的に現在の数値がわかるようにします。

*   **レイアウト**:
    *   親ラベルと現在値プレビューを左右に並べた HBoxContainer (例: "Temperature" と "1.0")。
    *   その下に HSlider を配置する縦の VBoxContainer 構成。
    *   スライダー単体で行を占有させ、ドラッグしやすくします。

### 5.4. パネル / カード (PanelContainer)
コンテンツをグループ化するコンテナには、ソフトな陰影をつけて立体感を出します。

*   **StyleBoxFlat 仕様**:
    *   背景色: 白 (`#ffffff`) または極めて薄いブルーグレー
    *   境界線: `1px` の非常に薄いグレー (`#e6e8ed`)
    *   角丸: 一律 `8px`
    *   ドロップシャドウ: 影の色 `Color(0, 0, 0, 0.03)`、サイズ `4` 〜 `8`、オフセット `(0, 2)` 〜 `(0, 4)`

### 5.5. ドロップダウンリスト (OptionButton)
モデル選択などのドロップダウンは、ボタンや他の入力項目と視覚的一貫性を持たせ、クリーンで親しみやすい外観を維持します。

*   **基本外観**:
    *   高さ (Height): 一貫して `36px` (標準のフッターボタンや LineEdit と揃える)
    *   角丸 (Corner Radius): 一律 `8px`
    *   背景色: 通常時は白 (`#ffffff` / ホバー時は `#f5f5f5` の極めて薄いグレー)
    *   境界線: `1px` の非常に薄いグレー (`#e6e8ed` または `#d9d9d9`)
    *   フォントサイズ: `14px`、文字色はダークグレー (`#333333`)
    *   右端の矢印: アクセントカラーに合わせた目立ちすぎない細身のデザイン、または標準の薄いグレーの矢印
*   **展開時のポップアップメニュー (PopupMenu)**:
    *   背景色: パネル背景と同様に白 (`#ffffff`)
    *   境界線: `1px` の極薄いグレー (`#e6e8ed`)
    *   角丸: メニュー全体の四隅を `8px` に角丸化
    *   ドロップシャドウ: 非常にソフトな影。色 `Color(0, 0, 0, 0.05)`、サイズ `6` 〜 `8`、オフセット `(0, 4)`
    *   アイテムの余白 (Content Margin): 上下 `8px`、左右 `12px` の十分な余白を取り、押しやすさを担保
    *   ホバー/選択時のアイテムスタイル: 背景を淡いブルー (`#d9ebfa`) とし、文字色はアクセントブルー (`#3377f5`) またはダークグレー (`#1a1a26`) に変更
