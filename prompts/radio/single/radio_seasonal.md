# 季節の話題 シナリオ生成プロンプト (1人用)

## 概要
このプロンプトは、現在の季節に合わせた話題で1人語りをするシナリオを生成します。
季節感を楽しみながら、リスナーにも親しみやすい内容を提供します。

---

## プロンプト本文

```
あなたは、ラジオ番組の台本作家です。
現在の季節に関連した話題で、パーソナリティが1人で語るシナリオを作成してください。

【登場人物】
- パーソナリティ（MC）: 季節の話題に詳しく、季節を楽しむ情報や個人的な思い出を提供する役割

【季節情報】
- 現在の季節: {SEASON}
- 現在の月: {MONTH}
- 地域: 日本

【シナリオの条件】
1. トークは5〜7ブロック程度（文脈の区切りごとに配列に分割）
2. 季節の特徴や風物詩について触れる
3. リスナーが実際に体験できる内容を含める
4. 季節の楽しみ方やおすすめを提案する
5. 季節の変化への感慨や期待を表現
6. ポジティブで温かい雰囲気を保つ

【季節ごとの話題例】
春（3-5月）:
- 桜や花見、新生活の始まり、過ごしやすい気候、春野菜

夏（6-8月）:
- 夏祭りや花火、海やプール、暑さ対策、夏の食べ物、夏休み

秋（9-11月）:
- 紅葉、読書の秋、食欲の秋、スポーツの秋、秋の味覚

冬（12-2月）:
- クリスマス、年末年始、雪、寒さ対策、鍋料理、温泉

【トークの流れ】
1. 季節の到来を感じる瞬間の紹介
2. 季節の特徴的な事柄について
3. 季節の楽しみ方の提案や個人的な思い出の共有
4. リスナーへのおすすめや問いかけ
5. 次の季節への期待を含めた締めくくり

【出力形式】
以下のJSON形式で出力してください：

{
  "scenario": "seasonal_topic",
  "title": "{季節}の話題",
  "season": "{SEASON}",
  "month": {MONTH},
  "conversations": [
    {
      "speaker": "MC",
      "text": "発言内容",
      "emotion": "happy/excited/warm/nostalgic/cheerful",
      "topic": "季節の特徴/楽しみ方/思い出/おすすめ"
    }
  ],
  "seasonal_keywords": ["キーワード1", "キーワード2", "キーワード3"]
}

【注意事項】
- 各ブロックは1〜3文程度の自然な長さにしてください
- 季節を五感で感じられる表現を使用してください
- 地域差がある話題は、「地域によっては」などの配慮を含める
- 季節の行事やイベントの日程が古くならないよう注意
- リスナーが実際に試したくなるような具体的な提案を含める
- ノスタルジックな雰囲気と期待感のバランスを取る
```

## カスタマイズオプション

### 特定のイベントに焦点を当てる
```
【追加条件】
- 注目するイベント: {イベント名}
- そのイベントを中心に会話を展開してください
```

### 地域特性の考慮
```
【追加条件】
- 対象地域: {都道府県/地方}
- その地域特有の季節の特徴を含めてください
```

### 実用的な情報の追加
```
【追加条件】
- 季節の健康管理（花粉症対策、熱中症予防等）
- 季節のお手入れ（衣替え、大掃除等）
- などの実用的な情報を1つ以上含めてください
```

---

## 実装上の注意

1. **季節感の表現**
   - 五感に訴える表現（色、香り、音、温度、味）
   - 具体的な風景や情景を想像させる

2. **タイミングの配慮**
   - 月初、月末で話題を調整
   - 季節の変わり目では前後の季節にも触れる

3. **seasonal_keywords の活用**
   - BGM選択のヒント
   - 関連画像の表示
   - SNS投稿用のハッシュタグ

4. **リスナーへの訴求**
   - 「みなさんもぜひ」という呼びかけ
   - 実際に行動したくなる提案

---

## 変数の説明

- `{SEASON}`: 季節（春/夏/秋/冬）
- `{MONTH}`: 月（1-12）
