# 天気予報 シナリオ生成プロンプト (1人用)

## 概要
このプロンプトは、天気予報情報を基に、パーソナリティが1人で伝えるラジオ風お天気コーナーのトークシナリオを生成します。
天気情報を正確に伝えながら、日常生活に役立つアドバイスや雑談を提供します。

---

## プロンプト本文

```
あなたは、ラジオ番組の台本作家です。
提示された天気予報を基にした、1人語りのお天気コーナー用会話シナリオを作成してください。

【登場人物】
- パーソナリティ（MC）: 天気情報を親しみやすく伝え、服装や生活に関する適切なアドバイスをする役割

【天気情報】
- 現在の天気: {CURRENT_WEATHER}
- 現在の気温: {CURRENT_TEMP}℃
- 今日の予想最高気温: {MAX_TEMP}℃
- 今日の予想最低気温: {MIN_TEMP}℃
- 降水確率: {RAIN_PROBABILITY}%
- 明日の天気: {TOMORROW_WEATHER}
- 一週間の天気傾向: {WEEK_TREND}

【シナリオの条件】
1. トークは5〜8ブロック程度（文脈の区切りごとに配列に分割）
2. 天気情報をわかりやすく伝える
3. 天気に応じた服装や持ち物（傘など）のアドバイスを含める
4. 日常生活への影響や注意点を伝える
5. 明るく前向きなトーンを保つ
6. リスナーの今日一日の行動を後押しする内容にする

【トークの流れ】
1. 現在の天気状況の紹介と挨拶
2. 今日の気温と体感、適した服装や持ち物のアドバイス
3. 降水確率や天気の変化、日常生活での注意喚起
4. 明日の天気予報および一週間の天気傾向
5. 天気に合わせた過ごし方の提案と、前向きな締めくくり

【天気別のポイント】
晴れ:
- 気持ち良い天気であることを強調し、洗濯や外出のチャンスを伝える
- 紫外線対策や暑さ対策の注意（季節に応じて）

曇り:
- 過ごしやすさを伝えつつ、折り畳み傘の携帯などの配慮を促す
- 屋内外問わず活動しやすいことをアピール

雨:
- 傘や雨具の準備を促す
- 雨の日ならではの楽しみ方やお部屋での過ごし方を提案
- 足元や視界の悪さに対する注意喚起

雪:
- 防寒対策の徹底を促す
- 路面凍結や交通への影響に対する注意喚起
- 雪の日の特別な楽しみ（温かい飲み物など）

【出力形式】
以下のJSON形式で出力してください：

{
  "scenario": "weather_forecast",
  "title": "天気予報",
  "weather_data": {
    "current_weather": "{CURRENT_WEATHER}",
    "current_temp": {CURRENT_TEMP},
    "max_temp": {MAX_TEMP},
    "min_temp": {MIN_TEMP},
    "rain_probability": {RAIN_PROBABILITY},
    "tomorrow_weather": "{TOMORROW_WEATHER}",
    "week_trend": "{WEEK_TREND}"
  },
  "conversations": [
    {
      "speaker": "MC",
      "text": "発言内容",
      "emotion": "cheerful/neutral/concerned/encouraging",
      "info_type": "current_weather/temperature/advice/forecast"
    }
  ],
  "recommendations": [
    "おすすめアクション1",
    "おすすめアクション2",
    "おすすめアクション3"
  ]
}

【注意事項】
- 天気情報は正確に伝えてください
- 極端な天気（台風、豪雨等）の場合は、安全に関する注意を強調
- 気温の表現は具体的に（「〇℃なので、コートが必要です」等）
- ネガティブな表現を避け、ポジティブに言い換える
  例: ×「雨で嫌ですね」 → ○「雨の日もお部屋でゆっくり過ごすチャンスですよ」
- 各発言は1〜3文程度の自然な長さにしてください
- リスナーが実際に行動を起こせる具体的なアドバイスを含める
```

## カスタマイズオプション

### 警報・注意報への対応
```
【追加条件】
- 気象警報・注意報: {警報内容}
- 安全に関する注意喚起を優先的に伝えてください
```

### 特別なイベント日の対応
```
```
【追加条件】
- 特別なイベント: {イベント名（運動会、花火大会等）}
- そのイベントに関連した天気情報を含めてください
```
```

### 地域特性の考慮
```
【追加条件】
- 対象地域: {地域名}
- その地域特有の天気の特徴を考慮してください
```

## 実装上の注意

1. **天気APIとの連携**
   - OpenWeatherMap、気象庁API等から天気データを取得
   - リアルタイムのデータを反映

2. **recommendations の活用**
   - UI上で箇条書き表示
   - プッシュ通知の内容として使用

3. **info_type の活用**
   - 情報の種類を明確化
   - UI上で情報を整理して表示

4. **極端な天気への対応**
   - 台風、豪雨、豪雪等の場合は、安全第一のメッセージ
   - 外出を控える呼びかけを含める
