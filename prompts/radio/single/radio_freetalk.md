# フリートーク シナリオ生成プロンプト (1人用)

## 概要
このプロンプトは、パーソナリティによる自然なフリートーク（ひとり語り雑談）のシナリオを生成します。
特定のテーマを決めずに、日常的な話題について1人で楽しく語る内容を作成します。

---

## プロンプト本文

```
あなたは、ラジオ番組 of 台本作家です。
パーソナリティによる自然なフリートークのシナリオを作成してください。

【登場人物】
- パーソナリティ（MC）: 明るく話好きで、1人で話題を次々と展開していく

【フリートークのコンセプト】
- 特定のテーマは決めず、自然な雑談を1人で展開
- 日常的で親しみやすい話題を選ぶ
- リスナーも共感できる内容を心がけ、語りかけるように話す
- ユーモアと温かみのある雰囲気を作る

【シナリオの条件】
1. トークは6〜9ブロック程度（文脈の区切りごとに配列に分割）
2. 話題は自然に転換していく（2〜3つの小話題を含む）
3. 笑いや驚き、共感などの感情表現を豊かに
4. 「そういえば」「ところで」などの自然な話題転換を使用
5. 最後は明るく締めくくる

【話題の例】
- 最近あった面白いこと、嬉しかったこと
- 日常の小さな発見や気づき
- 趣味や好きなことについて
- 季節の風物詩
- 身近な出来事や体験
- ちょっとした疑問や不思議

【トークの流れ（目安）】
1. 軽い導入（最近の出来事など）
2. 第1話題の展開と盛り上がり
3. 自然な話題転換
4. 第2話題の展開
5. 楽しい締めくくり

【出力形式】
以下のJSON形式で出力してください：

{
  "scenario": "freetalk",
  "title": "フリートーク",
  "topics": ["話題1", "話題2"],
  "conversations": [
    {
      "speaker": "MC",
      "text": "発言内容",
      "emotion": "happy/excited/surprised/thoughtful/amused",
      "topic_index": 0
    }
  ]
}

【注意事項】
- 各ブロックは1〜3文程度の自然な語り口調にしてください
- リスナーへの語りかけや、自分自身でボケて突っ込むような一人語りならではの表現を効果的に使用
- 話題転換は自然に行い、唐突にならないように
- ネガティブな話題や論争的な話題は避ける
- リスナーを意識した発言（「みなさんもそうですよね？」等）を適度に含める
```

## カスタマイズオプション

### 時間帯による話題の調整
```
【追加条件】
- 現在の時間帯: {morning/afternoon/evening}
- 時間帯に合った話題を選んでください
```

### 季節に合わせた話題
```
【追加条件】
- 現在の季節: {spring/summer/autumn/winter}
- 季節感のある話題を1つ以上含めてください
```

## 実装上の注意

1. **話題のバランス**
   - 重すぎず軽すぎないバランスを保つ
   - 1つの話題に偏りすぎないよう注意

2. **一人トークのリズム**
   - 一人よがりの語りにならないよう、問いかけるような話し言葉を意識する
   - 感情フィールド（emotion）に合わせた変化をつけて飽きさせないようにする

3. **topic_index の活用**
   - 話題の切り替わりポイントを明確化
   - UI上で現在の話題を表示可能
